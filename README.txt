AI Research Lab Website Project

Files Included:
1. index.html        -> Home page
2. about.html        -> About the Lab page
3. team.html         -> Team Members, Research Projects, Contact page
4. style.css         -> Main stylesheet
5. script.js         -> Contact form alert script

How to Run:
- Keep all files in the same folder.
- Open index.html in any web browser.

How to Host:
- Upload the folder to a GitHub repository.
- Go to repository Settings > Pages.
- Choose the main branch and root folder.
- Save to get a public GitHub Pages website link.

Project Notes:
- Built with HTML, CSS, Bootstrap, and a small JavaScript file.
- Responsive on desktop, tablet, and mobile screens.
- Uses fewer files for less complexity, matching the lab instruction.
